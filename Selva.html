Introduction

This projects represents a model of iOS App Store. App Store contains different apps and games and, of course, lots of data about them. To work with the App Store, a lot of things are required, such as price, rating, category, number of downloads, etc.


Methodology

To achieve my results, I have used a lot of different technologies, such as

  • XML
  • XML Schema
  • HTML
  • XSL
  • XSLT

Project contents

The project consists of several files:

XML File
Contains main data/dd>
XML Schema File
Describes the structure of XML
XSL Transforamtion File
Code used to generate HTML output
HTML File
HTML table (output) after XSLT

Elements

The root element is <appStore>. <appStore> contains, as required, 5 records of <application>.

Each <application> consists of different attributes and elements, such as

  • <title>
  • <developer>
  • <genre>
  • <size>
  • <price>
  • <rating>
  • <numberOfDownloads>
  • <numberOfViews>
  • <numberOfReviews>
  • <notifications>
  • <supportediOS>
  • <supportedDevices>
  • etc.

Transformation

To wrok with the data from XML, I used XSLT. It allows you to get different kinds of data from XML. For each record the same output can be requested and this output is organized in the form of table.
