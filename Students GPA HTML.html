Introduction

this project is representing highest GPA of 5 students from Informatics course of Czech University of life sciences.


Methodology – technologies used

As required in this project, I used:

  • XML
  • XML Schema
  • HTML
  • XSL
  • XSLT

Everything was validated by means of W3C Online Validator


Work content

This project consists of 5 files:

XML file
Comprises the main content of the project
XML Schema file
Describe the structure of the XML document above
XSL Transformation file
Style sheet for XML
HTML file
Contains the transforamtion from XML to an HTML table

Transformation description

XSL was used in order to transform XML file data into HTML table is showing a list of students sorted by their sur names.

Students with GPA better than 1.2 are highlited with green color, others with red.
